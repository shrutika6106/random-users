 🌐 Random Users React App

This project is a simple React-based web application that fetches and displays random user data using a public API.

🔗 GitHub Repository  
https://github.com/shrutika6106/random-users  

 🚀 Features

- Fetch random users from API
- Display user data in card format
- Show:
  - Profile image
  - Name
  - Email
  - City
- Load new users with a button
- Loading indicator while fetching data

🧠 Concepts Used

- React (useState, useEffect)
- API Fetching (fetch)
- JSON data handling
- Dynamic UI rendering using map()
- Basic CSS styling

 🔗 API Used
https://api.freeapi.app/api/v1/public/randomusers

 🖥️ Tech Stack

- React (Vite)
- JavaScript
- HTML
- CSS

📌 Purpose of Project

This project demonstrates how to:
-Fetch real-time data from an API
-Handle and display dynamic data in React
-Build a simple user interface using reusable components.
